# DeepSeek → Computer Wisdom → Zora Ingestion Workflow v1

Status: `CANDIDATE_WORKFLOW`
Operator: `jaywisdom.base.eth`
Mode: `PUBLIC_OSINT_ONLY`

## Purpose

This workflow converts Chinese-language DeepSeek strategic simulation outputs into replayable Computer Wisdom receipts before any public Zora posting.

The goal is not intelligence collection, targeting, or operational warfare. The goal is lawful public-source strategic analysis: claims become receipts, receipts become reviewable artifacts, and only verified artifacts become public narrative.

## Pipeline

```text
AIR_UNIVERSITY_BASELINE
  → CHINESE_DEEPSEEK_PROMPT
  → RAW_MODEL_OUTPUT
  → CLAIM_EXTRACTION
  → SOURCE_RECEIPT_GATE
  → COMPUTER_WISDOM_L2_CARD
  → HUMAN_REVIEW
  → PUBLIC_ZORA_INGESTION
```

## Core Law

```text
NO RECEIPT → NO CLAIM
NO SOURCE → NO PUBLIC POST
NO REPLAY → NO INGESTION
```

## Input Constraints

- Chinese-language prompt/output preferred for adversarial framing diversity.
- Public sources only.
- No clandestine collection.
- No instructions for sabotage, evasion, targeting, or weapons use.
- Every strategic claim must be marked as one of:
  - `VERIFIED_PUBLIC_SOURCE`
  - `PLAUSIBLE_NEEDS_SOURCE`
  - `SPECULATIVE_HOLD`
  - `REJECTED_UNSUPPORTED`

## Machine-Speed Stage

Computer Wisdom may process at machine speed, but promotion remains gated:

1. Parse raw DeepSeek output.
2. Extract claims.
3. Require source receipt per claim.
4. Score signal vs noise.
5. Emit L2 cards.
6. Hold for human review.
7. Publish only verified summary to Zora.

## Maxwell's Demon Filter

The filter separates high-value signal from narrative noise by testing:

- tempo evidence
- industrial capacity evidence
- launch cadence evidence
- procurement evidence
- academic/patent acceleration
- orbital infrastructure indicators
- contradiction across sources
- missing receipt risk

## Public Zora Rule

Zora posts are cultural/public summaries only. They must not imply classified access, official affiliation, or secret intelligence.

Preferred Zora format:

```text
Computer Wisdom L2: China Space Tempo Stress Test

Public OSINT only. Claims require receipts.
Signal: [one verified finding]
Receipt: [public source / hash / timestamp]
Status: HELD / VERIFIED / REJECTED

Verification over narrative. 🧾⚙️
```
