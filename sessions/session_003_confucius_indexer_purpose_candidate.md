## Vernacular Rule

Global language and local language are not the same.

The indexer should preserve:

- exact quote or phrase
- local meaning
- global translation
- uncertainty
- source path
- claim tag
- evidence requirement

## Investigative Workflow

1. Capture phrase, claim, code comment, or artifact.
2. Parse claims into PUBLIC / HELD / PENDING / REJECTED.
3. Identify local vernacular meaning.
4. Identify global/system meaning.
5. Link evidence or mark missing evidence.
6. Produce a Wisdom Report.
7. Create receipt if the report is preserved.
8. Batch or Merkle later only after approval.
