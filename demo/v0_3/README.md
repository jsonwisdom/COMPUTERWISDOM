# Constitutional Game Stack v0.3 Demo

This demo shows a bounded claim moving through the Constitutional Game Stack.

Flow:

1. Claim Input
2. Caseflow Transition
3. Goblin Verification
4. Agent Action Receipt
5. Replay Score Receipt
6. Anchor Hygiene Check
7. Final Replay Packet

Authority remains false throughout the entire flow.

Artifacts:

- demo_claim_001.json
- demo_replay_packet_001.json
- constitutional_game_stack_v0_3_runtime_demo.md

Purpose:

Provide a replayable example that demonstrates how receipts, gates, and transitions work without granting authority.
