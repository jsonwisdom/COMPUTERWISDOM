# Observation Protocol V0.1

## Purpose

Define how observations enter the system as ERS-compatible records.

Observations are records of witnessing.

Observations are not claims of truth.

## Core Invariant

Observation ≠ truth.

Observation ≠ adjudication.

Observation ≠ authority.

Observation = witnessed record with lineage.
