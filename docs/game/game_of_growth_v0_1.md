## Purpose

Game of Growth is the playable layer that sits underneath vectors and schemas.

The game converts public-record observation into quests, receipts, parody art, media cards, rule-maker response nodes, and replayable civic memory.

The system starts in Minnesota, then scales to other states, federal surfaces, and global public-record surfaces.

## Core Rule

```text
See the change. Save the proof. Skip the accusation.
```

## Game Loop

```text
QUEST
→ OBSERVE
→ SNAPSHOT
→ HASH
→ COMPARE
→ RECEIPT
→ PARODY_ART_UNLOCK
→ MEDIA_CARD
→ RULE_MAKER_RESPONSE
→ REPLAY
→ GROWTH
```

## Why This Exists

A small change in a public spreadsheet, PDF, dataset, budget, or contract can drift into consequences for many citizens.

The game makes the drift visible, funny, understandable, and replayable without claiming legal authority.

## Vector Layer

Vectors define the behavior of the game.

A vector can describe:

- what kind of public record is observed
- what kind of drift is detected
- what kind of receipt is generated
- what parody art can unlock
- what media card can be produced
- what rule-maker response is requested
- what replay test is required

Vector changes must happen through schemas, fixtures, or documented PRs.

No invisible vector mutation is allowed.

## Parody Art Unlock Rule

Parody art is enabled only after a receipt exists.

```text
NO RECEIPT → NO ROAST
RECEIPT EXISTS → PARODY MAY UNLOCK
```

Parody must target the observable absurdity of the record state, not private citizens or unsupported guilt claims.

Allowed:

- goblin receipt cards
- clown court badges
- changed scoreboard posters
- drift goblin stickers
- memory glitch trading cards

Blocked:

- harassment
- doxxing
- guilt declarations
- fraud declarations
- threats
- targeting private individuals

## First Minnesota Quest Examples

### Quest 001 — Budget Drift Hunt

Find two versions of a Minnesota budget-related public file. Hash both. Compare. Generate drift receipt.

### Quest 002 — Contract Memory Check

Find a public contract or procurement document. Compare an older archived version with the current version.

### Quest 003 — Spreadsheet Weight Change

Find a public CSV/XLSX where the byte size changed across versions. Receipt the old size, new size, old hash, and new hash.
