# Replayable Manic ($MANIC)

**Status:** Coherently Invalid until proven replayable.  
**Domain:** Constitutional observability for machine-speed governance.  
**Home:** Computer Wisdom protocol layer.

## What It Is

Replayable Manic ($MANIC) is a public constitutional observability protocol for replayable reasoning, public evidence, adversarial interpretation, and inspectable disagreement.

It is designed to reward **inspectability** and **replay fidelity** rather than certainty, popularity, engagement velocity, or narrative dominance.

## What It Is Not

$MANIC is not:

- a truth oracle
- a political scoring system
- a leak/archive system
- a legal adjudicator
- a settlement state machine
- an engagement farm
- a certainty market

## Core Invariant

**Inspectable disagreement is healthier than enforced coherence.**

The protocol does not answer:

> What is true?

It answers:

> Can this reasoning chain be replayed and challenged?

- Public-source only.
- Chronology is append-only.

- Confidence must be stated.
