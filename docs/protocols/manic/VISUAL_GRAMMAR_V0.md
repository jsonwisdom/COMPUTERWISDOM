# VISUAL_GRAMMAR_V0

**Replayable Manic – Visual & UI Grammar**  
**Version:** 0.0

## Core Principle

The interface must visually preserve **unresolved ambiguity** and make inspectable disagreement feel native. Certainty is conditional; completeness is illusory.

## Visual Grammar

| Element | Meaning | Visual Treatment |
|---------|---------|------------------|
| Solid lines | OBSERVED | High opacity, grounded weight |
| Dashed lines | INTERPRETED | Lighter, conditional |
| Fractured / glowing edges | DISPUTED | Preserved tension, non-destructive |
| Gold overlay | SUPERSEDED / Clean Correction | Positive, high-status signal |
| Confidence opacity | Stated confidence level | Full opacity = high confidence; translucent = low |
| Redaction fog / blur | Replay gaps | Visualizes absence without destroying structure |
| Branch divergence heat | Interpretation spread | Shows lawful disagreement density |

## Replay Graph Structure

- Immutable root (public source) at top/center
- Vertical primary timeline with horizontal branching
- Full graph always accessible (collapse optional for readability)
- Hover/click nodes reveal: tier, tags, contributor, confidence, sources
- Timeline scrubber for chronological replay
- “Coherently Invalid” watermark on syntheses

## Supporting UI Elements

- **Badges**: Correction Alchemist, Fork Guardian, Provenance Anchor, etc.
- **Contributor Profiles**: Correction history, epistemic momentum, decay/recovery signals
- **Opacity Index Dashboard**: Radar charts, version history, side-by-side forks
- **Color Palette**: Institutional deep blues/grays + glitch teal (activity) + warm gold (corrections) + soft red (disputed, non-aggressive)

## Cultural Signal

The system must visually and interactively reinforce:  
**“Inspectable disagreement is healthier than enforced coherence.”**

This grammar turns the interface into a teaching tool for constitutional cognition under machine-speed conditions.
