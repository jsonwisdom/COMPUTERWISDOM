## 1. Purpose

Define the machine-speed layered operating model for COMPUTERWISDOM and JSONWisdom AL.

This blueprint connects:

- L0 human meaning
- L1 GitHub direct receipts anchored to `jaywisdom.eth`
- L2 jurisdiction, reputation, and agentic economies anchored to `jaywisdom.base.eth`
- L3 cockpit and replay mission control
- L4–L9 scale layers for institutional, agentic, economic, and civic expansion

The purpose is scale without silent category promotion.

## 5. GitHub Direct Layer Rules

The L1 GitHub Direct Layer must preserve:

- branch name
- commit SHA
- PR number
- changed file list
- review state
- merge state
- source repository
- linked repository, if applicable

A GitHub commit proves that a repository state changed.
It does not prove that the contents are universally true.
