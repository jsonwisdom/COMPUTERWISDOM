# 001 — Memory Needs Keepers

**Memory type:** human  
**Authority:** false  
**Package:** librarians  
**Identity anchor:** jaywisdom.base.eth

Librarians are not gatekeepers of truth.

They are keepers of access, order, context, and return paths.

A library does not make a book true.
It makes the book findable.
It preserves the shelf, the index, the card, the trail.

JOY honors librarians because memory without retrieval becomes fog.

A receipt that cannot be found is almost the same as a receipt that never existed.

Librarians protect the path back.

Authority remains false.
